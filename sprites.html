Lunar Magic Sprite Memory List

Select the Sprite Memory for your level to see a list of available vanilla sprites.

You may also select individual Vanilla SP# files to see which sprites belong in each GFX file.

To insert a sprite in Lunar Magic, select the Sprite tool and then go to Edit > Insert Manual. Use the Obj code above (e.g. CD) in the box that pops up.

This is a tool for Lunar Magic to show what sprites are available for any selected Sprite Memory or specific vanilla GFX files. Useful for Super Mario World hacking.
